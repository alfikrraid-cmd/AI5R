# Pump Registry Module

Pump Registry adalah modul inti LTSA Brain untuk menyimpan master data pompa customer.

## Main Table

`ltsa_pumps`

## Required Fields

- tag_number
- area

## Status Values

- UNKNOWN
- ACTIVE
- STANDBY
- REPAIR
- INACTIVE

## Used By

- Seal Registry
- PM Planner
- CM Work Order
- Inspection History
- Billing Report
